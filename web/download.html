Download the Timber compiler v1.0.1

Source distribution

timberc-1.0.1.tar.gz

MacOS X binary distribution

MacOS X 10.4 & 10.5 (Intel).

Linux binary distribution

openSUSE 10.3 (X86-64).

Accessing the source code repository

You need to have darcs installed in order to do this. Then type

      darcs get http://code.haskell.org/timber

Reporting bugs

Please use the Trac system at http://trac.haskell.org/timber/.
